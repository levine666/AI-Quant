## 1. 产品概述

### 1.1 产品名称（建议）

**Indicator Lab** / **指标实验室**

### 1.2 一句话描述

在浏览器中选择股票与指标参数，实时计算 RSI、MACD、布林带、ATR 并重绘图表的轻量级量化分析工具。

### 1.3 目标用户

- 量化课程学员（TASK02 实验延伸）
- 需要快速验证指标参数效果的个人投资者
- 已有 CSV / AkShare 数据、希望可视化对比的分析者

### 1.4 核心价值

| 痛点 | 解决方案 |
|------|----------|
| Notebook 改参数需重跑单元格 | 侧边栏滑块/输入框，改参即重绘 |
| 多只股票需切换多个文件 | 统一股票选择器 + 数据索引 |
| 四个指标分散在多张图 | 单页仪表盘，指标 Tab 或分屏联动 |
| 参数含义不直观 | 每个指标附带说明与当前读数摘要 |

### 1.5 非目标（v1 不做）

- 实时行情推送、自动下单
- 用户登录与云端保存（可列为 v2）
- 复杂策略回测与信号统计
- 移动端原生 App（响应式 Web 即可）

## 2. 用户场景

### 场景 A：切换股票对比指标

> 用户想看中芯国际港股、比亚迪 A 股的长电的 RSI 走势差异。

1. 打开工具首页  
2. 在「股票选择」下拉框选择标的  
3. 默认加载近一年日线，展示四指标概览  
4. 切换另一只股票，图表与摘要同步更新  

### 场景 B：调节 RSI 周期观察敏感度

> 用户想知道 RSI(6) 与 RSI(14) 对超买信号的差异。

1. 选中目标股票  
2. 进入「RSI」面板，将 period 从 14 改为 6  
3. 点击「应用 / 自动重绘」  
4. 图表与「当前 RSI 读数」更新，可选对比叠加旧参数曲线（v1.1）  

### 场景 C：导出当前视图

> 用户需要将当前参数下的图表放入作业报告。

1. 调整参数至满意状态  
2. 点击「导出 PNG」或「复制当前参数链接」  
3. 下载图表或分享 URL（含 query 参数）

## 4. 功能需求

### 4.1 股票数据模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 股票列表 | 从 spec / JSON 配置加载可选标的 | P0 |
| 数据源 | v1：本地预生成 CSV；v1.1：浏览器端调 AkShare 需后端 | P0 / P1 |
| 区间选择 | 近 365 日 / 自定义起止日期 | P0 |
| 复权 | qfq / hfq / none，与 spec 一致 | P0 |
| 数据校验 | 行数、OHLC 合法性，异常 toast 提示 | P1 |

**初始股票池（与现有 spec 对齐）：**

| ID | 名称 | 市场 | 代码 | 数据文件 |
|----|------|------|------|----------|
| smic_hk | 中芯国际 | 港股 | 00981 | smic_hk_00981_daily.csv |
| smic_a | 中芯国际 | A股 | 688981 | smic_688981_daily.csv |
| byd | 比亚迪 | A股 | 002594 | daily_002594_qfq.csv |
| cyp | 长江电力 | A股 | 600900 | daily_600900_qfq.csv |

### 4.2 指标计算模块

| 指标 | 可调参数 | 默认值 | 输出 |
|------|----------|--------|------|
| RSI | period | 14 | 折线图 + 30/50/70 参考线 |
| MACD | fast, slow, signal | 12, 26, 9 | DIF/DEA 线 + 柱状图 |
| 布林带 | period, std_multiplier | 20, 2.0 | 价格叠加 上/中/下轨 |
| ATR | period | 14 | 折线图（港元/元） |

**计算要求：**

- 纯前端实现（与 `scripts/indicators.py` 逻辑一致，可移植为 JS）
- 参数变更后 **300ms 防抖** 自动重算重绘（可选「应用」按钮）
- 窗口期 NaN 不绘制，tooltip 跳过无效点

### 4.3 图表交互

| 交互 | 说明 |
|------|------|
| 缩放 / 平移 | dataZoom 联动（主图与指标图 X 轴同步） |
| Tooltip | 十字光标，同日展示 OHLC + 四指标值 |
| 图例 | 可点击隐藏系列 |
| 主题 | 深色（默认，延续 dashboard.html 风格） |

### 4.4 参数 presets（v1.1）

- 「默认」「短线」「长线」三套参数模板一键切换
- 用户自定义参数存 `localStorage`

## 6. 技术方案（规划）

### 6.1 推荐栈

| 层级 | 选型 | 理由 |
|------|------|------|
| 结构 | 单页 HTML + 原生 JS 或 Vue3 轻量 | 与现有 dashboard 一致，易部署 GitHub Pages |
| 图表 | ECharts 5 | 已有经验，K 线 / 多轴 / dataZoom 成熟 |
| 计算 | JavaScript 移植 `indicators.py` | 纯前端无需后端 |
| 数据 | 静态 CSV + PapaParse 或预打包 JSON | GitHub Pages 友好 |
| 规范 | 读取 `indicator_lab.spec.yaml` → 构建时转 JSON | 与 TASK02 spec 统一 |

### 6.2 数据流

```
[spec/stocks.json] + [data/raw/*.csv]
        ↓ 用户选择股票
   [OHLCV DataFrame in memory]
        ↓ 读取侧边栏参数
   [computeIndicators(ohlcv, params)]
        ↓
   [ECharts setOption × 4]
        ↓
   [MetricCard 更新 latest snapshot]
```

## 8. 指标参数 UI 详细设计

### RSI

```
┌─ RSI ──────────────────────────────┐
│ 周期 (period)  [====●====] 14      │
│                2 ─────────── 50    │
│ ⓘ 衡量涨跌强弱，>70 超买 <30 超卖   │
└────────────────────────────────────┘
```

### MACD

```
┌─ MACD ─────────────────────────────┐
│ 快线 (fast)     [12]               │
│ 慢线 (slow)     [26]               │
│ 信号线 (signal) [ 9]               │
│ ⓘ DIF 上穿 DEA 为金叉               │
└────────────────────────────────────┘
```

### 布林带

```
┌─ 布林带 ───────────────────────────┐
│ 周期 (period)        [20]          │
│ 标准差倍数 (k)  [====●====] 2.0    │
│                1.0 ─────── 3.0     │
└────────────────────────────────────┘
```

### ATR

```
┌─ ATR ──────────────────────────────┐
│ 周期 (period)  [====●====] 14      │
│ ⓘ 波动幅度（港元），可用于止损宽度   │
└────────────────────────────────────┘
```

**全局操作按钮：**

- `重置为默认` — 恢复 spec 默认参数  
- `自动重绘` 开关 — 开：改参即算；关：点「应用」才算

## 11. 风险与约束

| 风险 | 应对 |
|------|------|
| 纯静态页无法直接调 AkShare | v1 用预生成 CSV；v2 加轻量 API |
| 前后端指标算法不一致 | 单元测试对齐 Python / JS 输出 |
| 参数极端值导致图表异常 | 输入校验 + 范围 clamp |
| 大数据量卡顿 | 限制最大 500 交易日；降采样可选 |
