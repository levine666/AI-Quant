## 1. 产品概述

### 1.1 背景

TASK02 已在 Notebook 中完成中芯国际港股（00981）的 RSI、MACD、布林带、ATR 计算与可视化。下一阶段需要将其升级为**可交互的 Web 工具**，支持：

- 切换不同股票数据源
- 四个指标参数可调
- 参数变更后实时/手动触发重绘

### 1.2 产品定位

面向量化学习者的**单页分析工具（SPA）**：轻量、本地可运行、无需后端服务（Phase 1），后续可对接 AkShare 实时取数或预置 CSV。

### 1.3 目标用户

- 完成 TASK02 作业的学生
- 需要快速对比多只股票技术指标的研究者
- 希望理解「参数变化如何影响指标形态」的学习者

## 2. 核心功能

### 2.1 功能清单（MoSCoW）

| 优先级 | 功能 | 说明 |
|--------|------|------|
| **Must** | 股票选择 | 从预置列表或上传 CSV 切换数据源 |
| **Must** | 四指标计算 | RSI / MACD / BOLL / ATR |
| **Must** | 参数调节 | 每指标独立参数面板，支持重置默认值 |
| **Must** | 图表重绘 | 点击「应用参数并重绘」刷新全部图表 |
| **Must** | 指标说明 | 悬停/侧边栏展示指标含义与当前读数解读 |
| **Should** | 日期范围筛选 | 近 3M / 6M / 1Y / 全部 |
| **Should** | 最新读数卡片 | 顶部展示最新一日 OHLC + 四指标快照 |
| **Should** | 导出 | 下载带指标 CSV、导出 PNG |
| **Could** | 多股对比 | 同一图表叠加两只股票归一化曲线 |
| **Could** | 主题切换 | 深色 / 浅色 |
| **Won't（v0.1）** | 实时行情推送 | 暂不实现 |
| **Won't（v0.1）** | 自动交易信号 | 暂不实现 |

### 2.2 用户故事

1. **作为用户**，我希望从下拉框选择「中芯国际港股」，以便查看其技术指标。
2. **作为用户**，我希望将 RSI 周期从 14 改为 6，以便观察更敏感的超买超卖信号。
3. **作为用户**，我希望点击「重绘」后四张图同步更新，以便对比参数效果。

## 4. 数据设计

### 4.1 股票数据源（Phase 1）

| 来源 | 说明 |
|------|------|
| 预置 CSV | 项目 `data/raw/`、`data/processed/` 已落盘数据 |
| 预置列表 | 中芯国际港股 00981、中芯国际 A 股 688981、比亚迪 002594、长江电力 600900 |
| 用户上传 | 接受标准 OHLCV CSV（date, open, high, low, close, volume） |

### 4.2 指标参数默认值

| 指标 | 参数 | 默认值 | 可调范围 | 步进 |
|------|------|--------|----------|------|
| RSI | period | 14 | 2–50 | 1 |
| MACD | fast | 12 | 2–50 | 1 |
| MACD | slow | 26 | 5–100 | 1 |
| MACD | signal | 9 | 2–30 | 1 |
| BOLL | period | 20 | 5–60 | 1 |
| BOLL | std_multiplier | 2.0 | 0.5–4.0 | 0.1 |
| ATR | period | 14 | 2–50 | 1 |

**约束：** MACD `fast < slow`，否则提示错误。

### 4.3 计算规范

与 `scripts/indicators.py` 保持一致：

- RSI / ATR：Wilder 平滑（`ewm alpha=1/period`）
- MACD：标准 EMA
- BOLL：SMA + rolling std

前端使用 JavaScript 实现，或 Web Worker 调用预编译逻辑，保证与 Notebook 结果一致。

## 5. 交互设计

### 5.1 主流程

```
选择股票 → 加载 OHLCV → 默认参数计算 → 展示图表
     ↓
调节某指标参数 → 点击「应用并重绘」→ 重新计算 → 更新图表 + 读数卡片
     ↓
（可选）切换日期范围 / 导出 CSV+PNG
```

### 5.2 重绘策略

| 模式 | 说明 | v0.1 建议 |
|------|------|-----------|
| 手动重绘 | 点击按钮后计算 | ✅ 采用（性能可控、便于教学演示） |
| 防抖自动重绘 | 滑块停止 500ms 后自动算 | Phase 2 |
| 实时重绘 | 滑块拖动即算 | 不推荐（大数据量卡顿） |

### 5.3 状态与反馈

- 加载中：图表区 skeleton + 「正在计算指标…」
- 参数非法：参数面板红色提示，禁用重绘按钮
- 计算完成：读数卡片数字高亮闪烁 300ms
- 无数据：空状态插图 + 引导上传 CSV

### 5.4 键盘与无障碍

- `R`：重绘
- `Esc`：重置参数为默认
- 图表需支持 Tab 聚焦到参数控件
- 色盲模式：MACD 柱用蓝/橙替代红/绿（Phase 2）

## 6. 技术方案（规划，暂不实施）

### 6.1 技术栈建议

| 层级 | 选型 | 理由 |
|------|------|------|
| 结构 | 单 HTML + ES Module | 与现有 dashboard.html 一致，易部署 GitHub Pages |
| 图表 | ECharts 5 | 项目已有经验，金融图表成熟 |

| 样式 | CSS Variables + Flex/Grid | 深色金融风，与现有看板统一 |
| 数据 | Fetch 预置 JSON/CSV | 静态部署友好 |

## 7. 界面设计规范

### 7.1 布局（1440 × 900 参考）

- **顶栏** 56px：深色 `#121826`
- **左参数栏** 280px 固定，可折叠至 48px（仅图标）
- **右图表区** 自适应，最小宽度 720px
- **卡片圆角** 12px，边框 `#243049`
- **主色**：蓝 `#5b8def`（交互）、紫 `#7b1fa2`（RSI）、绿 `#00897b`（ATR）

### 7.2 图表区排列

| 顺序 | 图表 | 高度 |
|------|------|------|
| 读数卡片 | RSI / MACD / BOLL位置 / ATR | 80px |
| 图1 | 收盘价 + 布林带 | 320px |
| 图2 | RSI | 200px |
| 图3 | MACD | 240px |
| 图4 | ATR | 180px |

### 7.3 参数控件组件

- 数值参数：`slider` + `number input` 联动
- 每组指标可折叠（Accordion），默认全部展开
- 组标题右侧显示当前参数摘要，如 `RSI(14)`

## 9. 风险与约束

| 风险 | 应对 |
|------|------|
| JS 与 Python 指标结果不一致 | 建立对齐测试用例，对比同一 CSV 输出 |
| 大数据量卡顿 | 限制最大 500 交易日；重绘加 loading |
| 港股/A 股字段差异 | 数据加载层统一 Schema |
| 移动端体验 | v0.1 优先桌面；移动端参数栏改为底部抽屉 |

## 10. 成功指标

- 切换股票后 1s 内完成加载与首绘
- 参数修改重绘 < 500ms（250 行数据）
- 四指标结果与 Notebook 误差 < 0.01%
- 用户无需读代码即可完成一次完整分析流程
